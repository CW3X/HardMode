关于
HardMode 模块为服务器所有者提供了创建自己的 HardMode 的工具，无需编写一行 C++/Lua 代码，完全可以通过数据库实现。
模式编辑器
这个部分将帮助您创建您的第一个困难模式，在创建模式之前，您需要想出一个名称，选择一种颜色，描述和限制。
限制
要为您的模式启用限制，您需要创建一个位掩码并将其存储在
hardmode_modes
表的
限制
列中。

在下面勾选您想要使用的限制，并将位掩码结果复制到您的模式的
限制
表列中，或者生成相应的 SQL 并自行应用。
